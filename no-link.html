Ошибка при переходе по ссылке

Данная ошибка возникает только на страницах моего портфолио.
Она связана с отсутствием реальной ссылки на контактные данные челоека, чьи услуги Вас могли заинтересовать.
Вот небольшой перечень причин, по которым это могло произойти:

  • Заказчик попросил сохранение анонимности при публикации работы в портфолио
  • Я не поддерживаю идею заказчика по личным или иным причинам
  • Заказчик не предоставил необходимых ресурсов, чтобы добавить реальную ссылку

Также могут одновременно выпосняться несколько из приведённых условий.
